# Climb Frontend

Front-end React + TypeScript para as telas de login e dashboard.

## Como iniciar

1. Abra o terminal em `frontend`
2. Execute `npm install`
3. Execute `npm run dev`

## Telas implementadas

- `src/pages/Login.tsx`
- `src/pages/Dashboard.tsx`

O `App.tsx` alterna entre login e dashboard com um estado simples de autenticação.
